
# RoadSense AI - Final Merged Project

This package combines:

1. Professional Smart Road Monitoring Frontend
2. YOLO Road Damage Detection Backend

Included Features:
- Modern Dashboard UI
- Login/Register Pages
- Map Integration
- Report Page
- AI Detection Backend Structure
- Flask Backend Starter
- Upload Detection Structure

Basic Run:

Frontend:
npm install
npm run dev

Backend:
pip install flask ultralytics
python main_backend.py
